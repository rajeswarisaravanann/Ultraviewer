UltraViewer Signaling Server

This server manages host/viewer signaling for WebRTC offers, answers, ICE candidates, and session state. It also provides a health endpoint for monitoring.

Server Status

Use the health endpoint to verify server readiness from monitoring or startup scripts.

GET /health

Signaling Events

Socket.IO events supported by the server:

  • host-register
  • viewer-connect
  • offer
  • answer
  • ice
